Hoi, ik ben Thijs

Sinds mijn jeugd heb ik een passie voor software development. Mijn vaardigheden bevatten Java, C#, HTML, CSS, Javascript en SQL. Naast mijn interesse in software ontwikkelen, vind ik het leuk om games te spelen en te creëren in Unity. Ook vind ik het leuk om te karten, verder werk ik in mijn vrije tijd als teamleider bij Jumbo. Mensen omschrijven mij vaak als een enthousiaste en betrokken persoon. Mijn doel is om mijn huidige opleiding tot software developer succesvol af te ronden en vervolgens naar het HBO te gaan waarbij ik mijn kennis kan uitbreiden.
